## Overview

This project processes the **UCI Human Activity Recognition Using Smartphones Dataset** to create a tidy data set that contains the **average of each measurement for each activity and each subject**.

The script downloads the raw data, merges training and test sets, extracts relevant measurements, applies descriptive labels, and outputs a clean summary data file.

1. Place the script in your working directory.
2. Open R or RStudio.
3. Run:

```r

```

The script will:

* Download and unzip the dataset (if not already present)
* Process the data
* Create `final_data.txt` in your working directory

---

## What the Script Does

### 1. Loads Libraries

Uses `dplyr` for data manipulation.

### 2. Downloads and Reads Data

* Downloads the UCI HAR Dataset if not already present
* Reads:

  * Feature names
  * Activity labels
  * Subject IDs
  * Training and test measurement data

### 3. Merges Datasets

* Combines training and test sets into one data frame
* Adds subject IDs and activity codes

### 4. Extracts Mean and Standard Deviation Measurements

* Selects only variables containing `mean` or `std`

### 5. Applies Descriptive Activity Names

* Replaces numeric activity codes with descriptive labels such as:

  * WALKING
  * SITTING
  * STANDING

### 6. Labels Variables with Descriptive Names

* Expands abbreviations and improves readability:

  * `Acc` â†’ `Accelerometer`
  * `Gyro` â†’ `Gyroscope`
  * `t` â†’ `Time`
  * `f` â†’ `Frequency`
  * `mean()` â†’ `Mean`
  * `std()` â†’ `STD`

### 7. Creates Final Tidy Dataset

* Groups by:

  * `subjects`
  * `activity`
* Computes the **mean of each variable**
* Writes the output to:

```
final_data.txt
```
